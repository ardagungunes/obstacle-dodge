# Obstacle Dodge

A personal Unity project created to practice gameplay programming, player movement systems, physics interactions, and general game development fundamentals.

The objective of the game is simple: navigate through an obstacle-filled environment while avoiding hazards and surviving as long as possible. The project was developed as a hands-on exercise to strengthen my understanding of Unity's component-based architecture and gameplay scripting in C#.

## Features

* Camera-relative player movement
* Physics-based character control
* Rotating obstacles
* Falling obstacle hazards
* Projectile interactions
* Score tracking system
* Third-person gameplay

## Gameplay Systems

### Camera-Relative Movement

One of the core systems implemented in the project is camera-relative movement.

Features:

* Movement based on camera orientation
* Smooth directional control
* Separation of player input and world-space movement
* Improved player navigation in a 3D environment

### Physics-Based Gameplay

The project uses Unity's physics system to create interactive gameplay elements.

Implemented concepts include:

* Rigidbody-based movement
* Collision detection
* Trigger events
* Dynamic obstacle interactions

### Scoring System

A simple scoring system tracks player progress and survival performance throughout gameplay.
